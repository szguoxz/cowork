# Keyboard Shortcuts

Use keyboard shortcuts to work faster in Cowork.

## Chat Page

| Shortcut | Action |
|----------|--------|
| `Ctrl+Enter` or `Cmd+Enter` | Send message |
| `Escape` | Cancel active agentic loop |
| `Y` | Approve all pending tool calls |
| `N` | Reject all pending tool calls |

## General Navigation

Standard browser/app shortcuts work:

| Shortcut | Action |
|----------|--------|
| `Ctrl+R` or `Cmd+R` | Refresh the page |
| `Tab` | Navigate between elements |
| `Enter` | Activate focused button |

## Tips

### Quick Tool Approval

When you trust the AI's tool calls, press `Y` to approve all pending tools at once. This is faster than clicking each approval button.

### Canceling Operations

If the AI is in a loop or you want to stop processing, press `Escape` to cancel the current operation.

### Multi-line Input

To enter a multi-line message in the chat input:
- On some systems, `Shift+Enter` creates a new line
- Or paste multi-line text directly
