## Quick Start

1. **Install Cowork** - Download from releases or build from source
2. **Configure API Key** - Set your API key in Settings or via environment variable
3. **Start Chatting** - Ask Cowork to help with coding tasks

- **Discussions** - Ask questions and share tips
